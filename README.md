## Features

- 구현기능 1. Bootstrap과 Scss를 활용한 반응형 홈페이지 구현
- 구현기능 2. Swifer를 활용한 slide구현
- 구현기능 4. Cookie를 활용한 모달창 구현

- 구현기능 2. Kopis API를 활용하여 장르별 공연을 화면에 표시하는 기능 구현(async, await , fatch)
